# אפליקציית צ'אט עם GPT - תחום מוגבל

אפליקציה פשוטה בפייתון לצ'אט עם GPT מוגבל לתחום ספציפי.

## דרישות מקדימות

1. Python 3.7 ואילך
2. מפתח API של OpenAI (ניתן להירשם ב-https://platform.openai.com/)

## התקנה

1. התקן את החבילות הנדרשות:
```bash
pip install -r requirements.txt
```

2. הגדר את מפתח ה-API שלך כמשתנה סביבה:

**Windows (PowerShell):**
```powershell
$env:OPENAI_API_KEY="your-api-key-here"
```

**Windows (CMD):**
```cmd
set OPENAI_API_KEY=your-api-key-here
```

**Linux/Mac:**
```bash
export OPENAI_API_KEY="your-api-key-here"
```

## שימוש

הרץ את האפליקציה:
```bash
python server.py
```

### פקודות זמינות:
- הקלד את השאלה שלך כדי לקבל תשובה
- `יציאה` או `exit` - לסיום האפליקציה
- `ניקוי` או `clear` - לאיפוס היסטוריית השיחה

## התאמה אישית

### שינוי התחום

ערוך את הקובץ `server.py` ושנה את המשתנה `DOMAIN` לתחום הרצוי:

```python
DOMAIN = "עזרה בתכנות Python"  # שנה לכל תחום אחר
```

דוגמאות לתחומים:
- "ייעוץ רפואי כללי"
- "עזרה במתמטיקה"
- "למידת אנגלית"
- "מתכונים בישול"
- "עצות גינון"

### שינוי המודל

כברירת מחדל האפליקציה משתמשת ב-`gpt-3.5-turbo`. ניתן לשנות ב-`server.py`:

```python
model="gpt-4"  # או כל מודל אחר
```

## הערות חשובות

1. **עלויות**: OpenAI אינו לחלוטין חינמי, אך מציע קרדיטים ראשוניים למשתמשים חדשים
2. **מודל**: `gpt-3.5-turbo` הוא המודל החסכוני ביותר
3. **הגבלות**: האפליקציה מוגבלת ל-500 טוקנים לתשובה כדי לחסוך בעלויות
